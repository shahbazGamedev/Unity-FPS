Controls:
	Movement: WASD
	Camera Movement: Mouse movement
	Shoot: Left Click
	Open Door: E
	
Every map is randomly generated in an interesting way
	- random room selection
	- random enemy selection
	
To proceed to the next room you must kill all enemies in the current room

There is a treasure room!
